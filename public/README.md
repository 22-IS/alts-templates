Шаблон репозитория студента.

## Статичные файлы

`info.txt` - информация о лабораторной работе.

Формат:
```
<номер_лр>
<код_студента>
```

Пример:
```
4
88a3b0f7afb7d18e70842fd5cdd5f49f
```

## Динамические файлы

`result.txt` - информация о результате проверки. В случае успеха содержит одно число - баллы, 
в случае неудачи - сообщение с описанием ошибки.

Пример:
```
100
```
