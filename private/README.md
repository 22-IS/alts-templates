Шаблон репозитория преподавателя.

## Статичные файлы

`students.yaml` - информация о студентах.

Формат:
```yaml
<код>:
    name: <ФИО>
    group: <группа>
    variant: <вариант>
```

Пример:
```yaml
88a3b0f7afb7d18e70842fd5cdd5f49f:
    name: Иванов Иван Иванович
    group: 22-ХХ
    variant: 4
```

## Динамические файлы
`results.csv` - результаты проверки лабораторных работ. Генерируется при выполнении соответствующей задачи.

Формат:
```csv
<ФИО>  <группа>  <баллы_лр1>  <баллы_лр2>  <баллы_лрX>  <средний_результат>
```

Пример:
```csv
ФИО  Группа  ЛР1  ЛР2  Среднее
Иванов Иван Иванович  22-ХХ  80  100  90
```
